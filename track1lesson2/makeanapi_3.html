MAKE AN API

The goal of this lesson is to demonstrate a way to take scientific data and put onto the web as an API. To do this, we will use the NASA Data Portal, data.nasa.gov.

Export a Dataset Create a Firebase project Import the Json File Access your API

Import the Json File

We need to import data into database.

Therefore, click on '...' symbol to see the 'Import JSON' option.

A dialog will be showed.

Find your JSON file that you saved in step 1, clicking on Browse button.

After that, import the file.

Seconds after, we can see comets' data on the database.

At this moment, configure the authorization rule.

The example bellow allows access for everyone. This mode isn't recommended for an API in production.

For our lesson, we will use the code bellow as the rule:

{ "rules": { ".read": true, ".write": true } }
