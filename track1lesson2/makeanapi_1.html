MAKE AN API

The goal of this lesson is to demonstrate a way to take scientific data and put onto the web as an API. To do this, we will use the NASA Data Portal, data.nasa.gov.

Export a Dataset

The first step is accessing NASA'S DATA PORTAL to choose a Dataset.

Click on 'Data Catalog'.

After that, select 'Space Science' and 'Tabular Data', for this lesson.

Click on the first item: 'Near-Earth Comets'. We'll work with data of comets!

The Export button will show a list of options like OData and Print.

In Download area, we'll click on JSON.

It will be our JSON content.

Use right button to save it in a file through 'Save as' functionality.

The step 3 will use this file.
