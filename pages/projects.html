  • NASA Space Apps 2019

    "Simoonlation" helps pilots choose the best place to land on the Moon to avoid as much dust as possible. It shows the most dangerous places and trigger warnings. Just choose the place or coordinates and see all the info you need to land safely.
    Global Nominee
    here | challenge | github

  • Resource Telescope

    The tool focuses on the reuse of previous Space Apps solutions by new challenge participants. They can start their apps from a known path by using links and references on other projects.
    app | API | about

  • Searchnauts

    It presents charts including countries and cities of NASA Datanauts worldwide. Actually, I developed this website for finding Datanauts in Canada to create a Space apps project in 2018 :) Possible next steps: Adding names, careers, researches, and contacts.
    here

  • NASA Space Apps 2018

    "Rocket Followers" is a solution that presents rocket launches' details, its spaceports, transmissions, statistics, and reentries. Over a weekend, some Datanaut members and I created this website and lots of datasets and API integrations.
    here | challenge | github

  • Hubbot

    Chatbot that searches for NASA APIs, according to key words. This bot was created in Nodejs, a back-end language, by using the Motion.AI and Socrata platforms.
